# Python OOP Practice

This repository contains beginner examples of Object-Oriented Programming (OOP) in Python.

I used AI tools as a learning assistant to clarify concepts, troubleshoot errors, and explore different approaches to solving problems. This has helped make my learning journey faster and more effective while still writing and practicing the code myself. All code was written and practiced as part of my personal learning journey.

## Topics Covered

- Creating Classes
- Constructors (`__init__`)
- Object Methods
- Modifying Object State
- Multiple Objects
- Simple ATM Simulation

## Files

student_class.py – Basic class example with grading logic

book_class.py – Simple object example

laptop_class.py – Object attributes example

bank_account_basic.py – Demonstrates modifying object data

atm_simulation.py – Mini ATM system using OOP and loops
